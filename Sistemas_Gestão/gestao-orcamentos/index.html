Instale no celular para usar sem internet, igual um app normal:

Proteção de Dados (Backup)

Atenção: Seus dados são locais!

Para sua privacidade, as informações são salvas apenas no seu aparelho. Se você formatar o celular ou limpar o navegador, poderá perder seus dados.

Como fazer Backup?

  1. Acesse Configurações.
  2. Clique em Exportar Backup (JSON).
  3. Salve o arquivo no seu Google Drive ou WhatsApp.

Como Restaurar?

Se trocar de celular, basta baixar o arquivo de backup e usar a opção Importar Backup na mesma tela de configurações.
