Dúvidas Frequentes (FAQ)

Como faço para trocar de dispositivo?

Para usar em outro aparelho, basta fazer o backup no dispositivo antigo, enviar o arquivo para o novo e usar a função "Restaurar Backup" nas configurações.

Funciona sem internet?

Sim! O Gestão Fit foi projetado para funcionar 100% offline após a instalação. Você só precisa de internet para fazer backup na nuvem ou enviar mensagens via WhatsApp.

Como cadastrar novos instrutores?

Acesse a aba "Instrutores", clique no botão "Novo Instrutor" e preencha o nome e a porcentagem de comissão dele.
